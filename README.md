# 🎵 Full-Feature Music Player

## 🌟 About

This project is a full-feature music player web application developed using HTML, CSS, and JavaScript. It is designed to provide a smooth and interactive user experience for playing and managing audio files.

The application demonstrates core web development concepts such as DOM manipulation, event handling, and media control. It includes essential features found in modern music players and focuses on clean UI design and responsiveness.

---

## 📌 Description

The music player allows users to play, pause, and navigate between songs. It also displays song details such as title, artist, and cover image, along with real-time progress tracking.

---

## 🚀 Features

* ▶ Play / Pause functionality
* ⏭ Next and Previous controls
* 📊 Progress bar with seek option
* 🔊 Volume control
* 🎶 Playlist support
* 🔁 Auto-play next song
* 🖼 Display song title, artist, and cover
* 📱 Responsive design
